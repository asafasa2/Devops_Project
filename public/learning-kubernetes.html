📚 Module 1: Kubernetes Fundamentals

⏱️ 60 minutes Beginner

Learn Kubernetes architecture, core components, and essential concepts. Understand the control plane, worker nodes, and how they work together.

🛠️ Module 2: Cluster Installation

⏱️ 75 minutes Intermediate

Master cluster installation using kubeadm, configure networking with CNI plugins, and join worker nodes to create a production-ready cluster.

⚙️ Module 3: Workloads & Scheduling

⏱️ 80 minutes Intermediate

Understand pods, deployments, services, and advanced scheduling concepts including node affinity, taints, and tolerations.

📝 CKA Fundamentals Quiz

⏱️ 20 minutes Quiz

Test your understanding of Kubernetes fundamentals with this comprehensive quiz covering architecture, components, and basic operations.

🧪 CKA Cluster Setup Lab

⏱️ 120 minutes Hands-on Lab

Hands-on lab to set up a complete Kubernetes cluster using kubeadm, configure networking, and deploy test applications.

🎓 CKA Practice Assessment

⏱️ 120 minutes Practice Exam

Comprehensive practice exam simulating the real CKA exam format with performance-based tasks and time constraints.
