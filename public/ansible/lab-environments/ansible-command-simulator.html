Ansible Command Simulator

Practice Ansible commands in this interactive simulator. Type commands and see the expected output.

Exercise 1: Basic Inventory Commands

Learn how to work with Ansible inventory files and list hosts.

ansible@control:~$
Hint: Try using ansible --help to see available options, or ansible all --list-hosts to list all hosts.
Solution:
ansible all --list-hosts - Lists all hosts in inventory
ansible webservers --list-hosts - Lists hosts in webservers group
ansible-inventory --graph - Shows inventory structure as a graph

Exercise 2: Ad-hoc Commands

Practice running ad-hoc commands to gather information and perform simple tasks.

ansible@control:~$
Hint: Use the ping module to test connectivity: ansible all -m ping
Solution:
ansible all -m ping - Test connectivity to all hosts
ansible all -m setup - Gather facts from all hosts
ansible webservers -m service -a "name=nginx state=started" - Check nginx service status

Exercise 3: Playbook Commands

Learn how to run and validate Ansible playbooks.

ansible@control:~$
Hint: Use ansible-playbook --syntax-check playbook.yml to validate syntax.
Solution:
ansible-playbook --syntax-check site.yml - Check playbook syntax
ansible-playbook site.yml --check - Dry run (check mode)
ansible-playbook site.yml - Execute playbook
ansible-playbook site.yml --tags webserver - Run only webserver tasks
